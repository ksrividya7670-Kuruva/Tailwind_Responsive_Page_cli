# 🚀 Responsive Landing Page

A modern and responsive landing page built using **HTML** and **Tailwind CSS**. The project demonstrates responsive layouts, utility-first styling, gradient backgrounds, navigation menus, hero sections, feature cards, pricing, testimonials, and a responsive footer.

## 📌 Features

- Responsive Navigation Bar
- Mobile-Friendly Layout
- Hero Section with Call-to-Action Buttons
- Feature Cards
- Pricing Section
- Testimonials
- FAQ Section
- Contact Section
- Responsive Footer
- Gradient Backgrounds
- Hover Effects
- Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/landing-page.git
```

2. Navigate to the project folder.

```bash
cd landing-page
```

3. Install Tailwind CSS (if not already installed).

```bash
npm install
```

4. Start Tailwind CSS.

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
```

5. Open `index.html` in your browser.
